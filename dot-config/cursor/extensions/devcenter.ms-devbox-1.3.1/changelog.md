# Change Log

All notable changes to the "dev box" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.1] - 2024-11-12

### Added
- Minor bug fixes

## [1.3.0] - 2024-09-17

### Added
- Copilot chat agent '@devbox' enabled(preview).

## [1.2.7] - 2024-04-17

### Added 
- Schema validation for userTasks.
- Support for imagedefinition.yaml files.

## [1.2.6] - 2024-04-10

### Fixed
- Dev Box output now opens automatically.
- Fixed insert snippets status message.


## [1.2.5] - 2024-04-05

### Fixed
- Telemetry fixes and improvements.
- Improved error messaging when attempting to run the extension outside of a Dev Box.
- Fixed success messaging when applying customizations succeeds.


## [1.2.4] - 2024-03-11

### Added
- Button to insert code snippets from examples.
- Button to collapse and expand all tasks.
- Telemetry. 


## [1.2.3] - 2024-01-22

### Fixed
- Compatibility with new schema

### Added
- Display tasks in copilot chat flow.
- Add toast while tasks are being loaded.

### Removed 

- Feature to create workload.yaml files.

## [1.2.2] - 2024-01-22

### Fixed

- Fixed schema incompatibility
- Accessibility improvements through key bindings and focus control.

## [1.2.0] - 2024-01-18 

### Added 

- Copilot chat integration for Dev Box customization.
- Localization. 

## [1.1.0] - 2023-12-10

### Changed 

-  UI improvements to the Task List view.

### Fixed 

- Fix dark and light hight contrast themes.
- Fix error when trying to list tasks without a workspace open.
