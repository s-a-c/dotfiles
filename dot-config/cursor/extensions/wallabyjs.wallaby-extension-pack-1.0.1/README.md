# Wallaby Extension Pack

This extension pack contains all of the Wallaby VS Code Extensions for improving your 
JavaScript / TypeScript development productivity:

* [Wallaby.js](https://marketplace.visualstudio.com/items?itemName=WallabyJs.wallaby-vscode)
* [Quokka.js](https://marketplace.visualstudio.com/items?itemName=WallabyJs.quokka-vscode)
* [Console Ninja](https://marketplace.visualstudio.com/items?itemName=WallabyJs.console-ninja)

## Wallaby.js

Wallaby.js is a developer productivity tool that runs your JavaScript and TypeScript tests immediately as you type, highlighting results in your IDE right next to your code.

![Wallaby.js in VS Code](https://wallabyjs.com/assets/img/vsc.gif)

## Quokka.js

Quokka.js is a developer productivity tool for rapid JavaScript / TypeScript prototyping. Runtime values are updated and displayed in your IDE next to your code, as you type.

![Quokka.js in VS Code](https://quokkajs.com/assets/img/main-video.gif)

## Console Ninja

Console Ninja is a VS Code extension that displays `console.log` output and **runtime errors** directly in your editor from your running browser or node application.

![Console Ninja in VS Code](https://console-ninja.com/images/docs-main.gif)
